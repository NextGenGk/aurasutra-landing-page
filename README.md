# AuraSutra Landing — Next.js

A pixel-perfect clone of MediZen redesigned as **AuraSutra**, built with Next.js 14 (App Router, no src folder), Framer Motion, and Tailwind CSS.

## Features

- ✅ Custom two-layer magnetic cursor
- ✅ Animated preloader with letter stagger
- ✅ Smooth scroll (Lenis)
- ✅ Scroll progress bar
- ✅ Animated Navbar (shrinks on scroll + blur)
- ✅ Hero section with SVG squiggle underline, floating badges, ripple button
- ✅ About section with rotating decorative rings
- ✅ Features section with hover-swap panel
- ✅ Infinite marquee ticker (pauses on hover)
- ✅ Why Choose Us with animated CountUp stats
- ✅ Appointment booking form with parallax bg
- ✅ Projects grid with hover overlay
- ✅ Special Care dark green card
- ✅ Testimonials carousel with slide transitions
- ✅ Blog asymmetric grid (1 large + 3 small)
- ✅ Newsletter section with submit success state
- ✅ Footer with social icons

## Getting Started

```bash
cd aurasutra-landing-v1
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- **Next.js 14** (App Router)
- **Framer Motion** — animations
- **Lenis** — smooth scroll
- **react-intersection-observer** — scroll triggers
- **react-countup** — animated stat counters
- **Tailwind CSS** — utility styles
